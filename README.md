# Shion Park

문제를 구조화하고 시스템화하는 프론트엔드 개발자입니다.  
운영 효율성과 사용자 경험을 함께 고려한 UI/UX 설계에 강점을 갖고 있으며, 기획·디자인·개발을 아우르는 협업 환경에 익숙합니다.

React, TypeScript 기반 SPA 아키텍처와 모노레포, 디자인 시스템, 상태 관리 최적화 경험을 바탕으로  
서비스의 일관성과 확장성을 고민하며 개발합니다.

## 프로젝트 경험

### Gymlight Manager — 피트니스 운영 플랫폼 (외주 프로젝트)

회원, 상담, 상품, 락커, 출결 등 헬스장 운영 기능을 전산화한 SPA 프로젝트입니다.
기획·개발 전반을 리딩하며 협업 프로세스와 코드 구조 모두 개선했습니다.

- React + TypeScript 기반 SPA, 모노레포 구조 및 디자인 시스템 적용
- Recoil, React Query 기반 상태 및 API 구성 / RBAC 기반 권한 UI 설계
- 지도 시각화(Kakao Maps)와 통계 차트(ApexCharts) 도입
- 락커 자동 배정 UI 개선 → 운영자 혼선 해소 및 평균 시간 16.7% 단축
- Figma 시안 및 기능 명세서 작성 → 클라이언트 커뮤니케이션 원활화
- PR/이슈 템플릿, 커밋 컨벤션, GitHub Actions + Cloudtype 기반 CI/CD 자동화
- QA 테스트 문서화 및 일정 관리 경험 보유
